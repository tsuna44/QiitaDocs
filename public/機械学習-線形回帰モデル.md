#線形回帰モデル
教師あり学習。

一次直線で値が予想できるもの。
要するに線型結合するもの。

```math
\begin{align}
\hat{y} &= w^Tx + w_0 \\
&= \sum_{j=1}^mw_jx_j + w_0
\end{align}
```

$w_j$の最適値を求める。

```math
y = w_0 + w_1x_1 + \epsilon
```

$y$:目的変数
$w_0$ : 切片
$w_1$ : 回帰係数
$x_1$ : 説明変数
$\epsilon$ : 誤差

切片と回帰係数を機械学習で求める。
具体的には、最小二乗法により推定する。

##最小二乗法
線形回帰モデルのパラメータは最小二乗法で求める。
予測値と実際の値の差が最小になるようにしていきます。

二乗誤差

【注】^ は予測値という意味です。

```math
\sum_i(\hat{y_i}-y_i)^2 = \sum_i\epsilon_i^2\\
```

iの総和を取ると言うこと。　二乗するとマイナスを考えなくても良くなる。

```math
\hat{y_i}-y_i = \epsilon_i
```

平均二乗誤差(二乗損失)のことを、MSE と言います。

二乗損失は一般的に外れ値に弱い。
$\rightarrow$ Huber損失、Tukey損失と言うものを使うと良い。

平均二乗誤差は下記式になる。

```math
MSE_{train} = \frac{1}{n_{train}}\sum_{i=1}^{n_{train}}(\hat{y}_i^{(train)} - y_i^{(train)})^2
```

この値を最小になるように求めていく。
最小値を求めるためには、勾配が 0 になる点を探す。
MSEの式を微分して計算する。

どう言うことかというと、下記になる値を計算する。
wで偏微分すると MSEは w の一次関数になります。
欲しいのは w です。

```math
\frac{\partial}{\partial w}MSE_{train} = 0
```

```math
\frac{\partial}{\partial w}MSE_{train} = 0\\
\rightarrow \frac{\partial}{\partial w} \left\{ {\frac{1}{n_{train}}\sum_{i=1}^{n_{train}}(\hat{y}_i^{(train)} - y_i^{(train)})^2} \right\} = 0
```

$\hat{y}^{(train)}$を展開する。

```math
\frac{\partial}{\partial w} \left\{ {\frac{1}{n_{train}}\sum_{i=1}^{n_{train}}(x_i^{(train)}w_i - y_i)^2} \right\} = 0\\
```

して、さらにX,Y,Wにして、行列表現すると。。。

```math
\frac{\partial}{\partial w} \left\{ \frac{1}{n_{train}}(XW-Y)^T(XW-Y) \right\} = 0
```

具体的な行列で記述すると、(XW-Y)は下記のようになる。。

```math
XW-Y = \left (
\begin{array} {ccccc}
x_{11} & \cdots & x_{1i} & \cdots & x_{1m}\\
\vdots & \ddots & & & \vdots \\
x_{i1} & & x_{ii} & & x_{im} \\
\vdots & & & \ddots & \vdots \\
x_{n1} & \cdots & x_{ni} & \cdots & x_{nm}
\end{array} \right )

\left (
\begin{array} {c}
w_{1}\\
\vdots\\
w_{i}\\
\vdots\\
w_{m}
\end{array} \right )
-
\left (
\begin{array} {c}
y_{1}\\
\vdots\\
y_{i}\\
\vdots\\
y_{n}
\end{array} \right )
=0
```

さらに、X,Y,Wの状態で偏微分の式を展開すると。。。

```math
\rightarrow \frac{1}{n_{train}}\frac{\partial}{\partial w}
\left\{ (W^TX^TXW - W^TX^TY - Y^TXW + Y^TY) \right \} = 0\\
\rightarrow \frac{1}{n_{train}}\frac{\partial}{\partial w}
\left\{ (W^TX^TXW - 2W^TX^TY + Y^TY) \right \} = 0\\
```

そして、Wで偏微分します。

```math
\rightarrow \frac{1}{n_{train}}
\left\{ 2X^TXW - 2X^TY \right\} = 0
```

【注】偏微分は一般的に下記のようなことになります。なるです。はい。
Aを対称行列として。。。

```math
\begin{align}
\frac{\partial(w^Tx)}{\partial w} &= x\\
\frac{\partial(w^TAw)}{\partial w} &= (A+A^T)x = 2Ax
\end{align}
```

ここで、欲しいのは wなので、$\frac{1}{n_{train}}$は気にしない。

```math
\left\{ 2X^TXW - 2X^TY \right\} = 0\\
\rightarrow X^TXW = X^TY\\
\rightarrow (X^TX)^{-1}(X^TX)W = (X^TX)^{-1}X^TY\\
\rightarrow W = (X^TX)^{-1}X^TY\\
```

Wはこれで計算できるようになりました。
wとYの予測値は、下記のようになる。

```math
\hat{W} = (X^TX)^{-1}X^TY\\
\hat{Y} = X_*\hat{W} = X_*(X^TX)^{-1}X^TY\\
```

【注】$X_*$ は予測したい新たな入力点です。

最尤法(もっともらしい値を求める)を使って回帰係数の推定をします。
最尤法を使うと、何が良いかというと。。。
○誤差を正規分布に従う確率変数を仮定し尤度関数の最大化を利用した推定も可能
○線形回帰の場合には、最尤法による解は最小二乗法の解と一致
