#主成分分析 (PCA)
変量の個数を減らすことに伴う、情報の損失はなるべく小さくする。

係数ベクトルが変わると、線形変換後の値が変化する。
線形変換後の変数の分散が最大となる射影軸を探索する。
要するに、たくさんの値を貫く直線が良いよね。という平たい理解をしてみた。

主成分分析の場合、下記の $Var(s_j)$を最大にする。

```math
\begin{align}
Var(s_j) &= \frac{1}{n}s_j^Ts_j\\
&=\frac{1}{n}(\bar{X}a_j)^T(\bar{X}a_j)\\
&=\frac{1}{n}a_j^T\bar{X}\bar{X}a_j\\
&=a_j^TVar(\bar{X})a_j
\end{align}
```

具体的には、ラグランジュ関数を最大にする。

```math
E(a_j) = a_j^TVar(\bar{X})a_j - \lambda (a_j^Ta_j -1)
```

$a_j^TVar(\bar{X})a_j$ : 目的関数
$\lambda$ : ラグランジュ乗数
$(a_j^Ta_j -1)$ : 制約条件

最適解は、ラグランジュ関数を微分して最適解を求める。

```math
\frac{\partial E(a_j)}{\partial a_j} = 2Var(\bar{X}a_j) - 2\lambda a_j =0\\
\rightarrow Var(\bar{X}a_j) = \lambda a_j
```

分散共分散行列は正定値対称行列。
固有値は０以上、固有ベクトルは直行になる。

##求め方
- 分散共分散行列を計算する
- 固有値問題を解く
- 最大m個の固有値と固有ベクトルのペアが出現する

めでたし、めでたし。

k番目の固有値を昇順に並べて、対応する固有ベクトルを「第k主成分」と呼ぶ。

##寄与率
第k主成分の分散の全分散に対する割合(第k主成分が持つ情報量の割合)

```math
c_k = \frac{\lambda_k}{\sum_{i=1}^m \lambda_i}
```

$\lambda_k$ : 第k主成分の分散
$\sum_{i=1}^m \lambda_i$ : 主成分の総分散

##累積寄与率
第1-k主成分まで圧縮した際の情報損失量の割合

```math
r_k = \frac{\sum_{i=1}^k \lambda_i}{\sum_{i=1}^m \lambda_i}
```

$\sum_{i=1}^k \lambda_i$ : 第1〜k主成分の分散
$\sum_{i=1}^m \lambda_i$ : 主成分の総分散
