#ロジスティック回帰モデル
分類問題(クラス分類)で使う。
教師あり機械学習モデル。
シグモイド関数を使う。

##シグモイド関数
シグモイド関数の微分は、シグモイド関数自身で表現することが可能なので、尤度関数の微分を行う際に計算が容易になる。

```math
\sigma(x) = \frac{1}{1+exp(-ax)}
```

xで偏微分すると。。。

```math
\frac{\partial \sigma(x)}{\partial x}
= \frac{\partial}{\partial x} \left( \frac{1}{1+exp(-ax)} \right ) \\
= a\sigma(x)(1-\sigma(x))
```

##ロジスティック回帰モデルの式

```math
P(Y=1|x)＝\sigma(w_0 + w_1x_1)
```

P(Y=1|x) : データが与えられたときにY=1になる確率
$w_0$ : 切片
$w_1$: 回帰係数
$x_1$ ：説明変数

学習で決めるのは切片と回帰係数。
その他は入力データ。

##最尤推定（さいゆう推定）
一番尤もらしい値を探す方法。
確率分布は色々あるけど、ロジスティック回帰はベルヌーイ分布を使う。

```math
E(w_0, w_1...,w_m) = -logL(w_0, w_1...,w_m)\\
= - \sum_{I=1}^n \{ y_i logp_i + (1-y_i) log(1-p_i) \}
```

サンプル数が少ないと過学習を起こす傾向がある。


##勾配降下法
反復学習によりパラメータを逐次的に更新するアプローチの一つ

```math
w(k+1) = w^k - \eta \frac{\partial E(w)}{\partial w} 
```

k+1 の w は、k番目の w に $- \eta \frac{\partial E(w)}{\partial w}$を足すと推測できる。

$\frac{\partial E(w)}{\partial w}$は下記の式になる。

```math
\frac{\partial E(w)}{\partial w} =
\sum_{I=1}^n \frac{\partial E_i(w)}{\partial p_i} \frac{\partial p_i}{\partial w}\\
= -\sum_{I=1}^n(y_i -p_i)x_i
```

だから、計算式を合わせると。。。

```math
w(k+1) = w^k + \eta \sum_{I=1}^n(y_i -p_i)x_i
```


パラメータが更新されなくなったら、勾配が0になったということ。

##確率的勾配効果法
データをランダムに選んでパラメータを更新する。

```math
w(k+1) = w^k + \eta (y_i - p_i)x_i 
```

確率的勾配降下法なら、なんだったら手計算できるかも。。。

##分類の評価方法
- True Positive
- False Positive (正常を間違えて異常)
- True Negative
- False Negative (異常を間違えて正常)

- 再現率(Recall)
- 適合率(Precision)
- F値

100%正確な分類というのはやっぱり難しいのかなぁと。
でも、正確性って人間が客観的に判断するんだろうなぁと。
