#非線形回帰モデル
大抵のケースはデータを線形で捉えられるようなことは少ない。
非線形で捉えられるようにする。

```math
\hat{y} = w_0 + w_1\phi_1(x) + w_2\phi_2(x) ...
```
非線形といっても、線形モデルの x が $\phi(x)$ に変わるだけ。
そして、w(重み)に着目すると線形なんです。

###基底展開法
回帰関数として、基底関数と呼ばれる既知の非線形関数とパラメータベクトルの線型結合を使用する。
要するに、基底関数をいくつか足し合わせるといい感じの曲線を作ることができるわけで。

####基底関数
- 多項式関数

```math
\phi_j = x^j
```

- ガウス型基底関数

```math
\phi_j(x) = exp\left\{ - \frac{(x-\mu_j)^2}{2h_j} \right\}\\
下記のようにも記述する。\\
\phi_j(x) = exp\left\{ - \frac{(x-\mu_j)^2}{\sigma^2} \right\}
```

${2h_j}$, $\sigma^2$は２つの山の頂点の距離をコントロールするというイメージ
ガウス型関数の2つの山の頂点の幅(広がり)を学習によって狭めていくイメージ。


- スプライン関数/Bスプライン関数

省略します。

####未学習と過学習

○未学習とは？
学習データに対して、十分小さな誤差が得られないモデル
■(対策)モデルの表現力が低いため、表現力の高いモデルを利用する

○過学習とは？
小さな誤差は得られたけど、テスト集合誤差との差が大きいモデル
■(対策1) 学習データの数を増やす
■(対策2) 不要な基底関数(変数)を削除して表現力を抑止
具体的には
・基底関数の数や位置、バンド幅を減らす。
多けりゃいいってもんではない。
・適切な基底関数を用意する。
なんでもいいってもんではない。

■(対策3) 正則化法を利用して表現力を抑止する。
算出された値を補正してあげる。
式で表現すると下記のようになる。

```math
S_\gamma = (y - \phi w)^T(y -\phi w) + \gamma R(w)
```

$\gamma R(w)$が罰則。
$\gamma$ は係数。
今までと同じ書き方をすると。。。

```math
E(w) = MSE + \lambda w^Tw
```

目的はMSEを最小化したい。罰則でうまく制御する

####正則化項
- 正則化項がない -> 最小二乗推定量
- L2ノルム(丸)を使用 -> Ridge推定量。いくつかのパラメータは限りなく0に近づく。
- L1ノルム(四角形)を使用 -> Lasso推定量。いくつかのパラメータが0になる。


$\lambda w^Tw$ が罰則

縮小推定：パラメータを0に近づけるように検定する。
スパース推定：いくつかのパラメータを0に近づけるようにする。
